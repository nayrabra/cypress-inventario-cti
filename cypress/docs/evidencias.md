# Evidências

As evidências dos testes foram registradas por meio de gravações utilizando a ferramenta **Loom**, contemplando tanto testes manuais exploratórios quanto testes automatizados com Cypress.

Os testes foram executados com base nas histórias de usuário, critérios de aceitação e cenários definidos na documentação do projeto.
